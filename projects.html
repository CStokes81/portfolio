My Projects

Word Jumble

This app was created to show I had an understanding of React while attending Digital Crafts. This game gives the user scrambled words and points for each correct word that is correctly unscrambled.

  • Used Postgres & Sequelize to create a backend database whereas users can login & register to play the game.
  • Used React as a base in the development of this application and for building the User Interface (UI).

The Memory Game

This app was created because I like matching games and I thought it would be neat to create an app as such. This game gives the user the ability to find a matching card.

  • Used React as the base for this application and for the User Interface.
  • This app was deployed using Netlify which was a joy because everytime changes were made, they were automatically deployed.

The Shape Connection

This app was a ploy on one of my favorite childhood games, Tetris. This game allows the user to connect shapes and clear lines in order to level up. This app is still in progress...

  • Used React/Redux as the base for this application and for the User Interface.
  • Used Postgres & Sequelize to develop a backend that also hashed user passwords upon logging in.
